========================
3) MARCAR PARCELAS PAGAS POR MIM (CASO SEJA PARA OUTRA PESSOA)
========================

Problema:
Às vezes eu pago a parcela no cartão,
mas a pessoa ainda não me pagou.

Solução:

Cada parcela precisa ter dois status separados:

{
  statusCartao: "pago" | "pendente",
  statusPessoa: "pago" | "pendente",
  dataPagamentoCartao,
  dataPagamentoPessoa
}

Regras:

- Se statusCartao = pago → impacta minha fatura normalmente
- Se statusPessoa = pendente → entra como valor que tenho a receber
- NÃO impactar negativamente minha saúde financeira
- Criar indicador visual:
  "Valor pago por você, aguardando reembolso"
